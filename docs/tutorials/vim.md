# Vim

**Vim** (/vÉªm/; a contraction of Vi IMproved) is a clone of Bill Joy's vi editor for Unix. It was written by Bram Moolenaar based on source for a port of the Stevie editor to the Amiga and first released publicly in 1991.

## Configuration

To open configuration file for vim type: `vi ~/.vimrc`. The simple configuration can look like this:

    " Set syntax on
    syntax on 
    
    " Turn on line numbering. Turn it off with "set nonu"
    set number
